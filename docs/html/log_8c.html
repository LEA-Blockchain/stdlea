void lea_log ( const char *  message)

Logs a message by copying it to a global buffer and passing it to the host.

Parameters
messageThe null-terminated string to log.
Note
This function uses a global buffer lea_log_buffer. If the message is longer than the buffer, it will be truncated. The buffer is cleared after the log message is passed to the host. A NULL message is treated as an empty string.
